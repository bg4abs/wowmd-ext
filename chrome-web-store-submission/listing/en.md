## Detailed description

wowMD is a lightweight Chrome extension for reading long public GitHub README and Markdown pages with less friction.

When a supported GitHub document is complex enough, wowMD shows a Better View button near the document. Open it to get a structured outline, jump between sections, fold long H2 sections, and continue in a full-screen reader tab when you want more space.

Key features:

- Shows up only on useful GitHub Markdown pages.
- Starts with an outline so you can understand the document structure first.
- Lets you jump from outline items to the matching section.
- Opens a local full-screen reader for longer reading sessions.
- Includes a small toolbar popup that explains when Better View appears and links to wowmd.app.
- Supports H2 folding, readable code blocks, wide tables, and relative README images.
- Sanitizes rendered HTML before inserting it into the page.
- Avoids accounts, tracking, ads, analytics, AI processing, and remote document upload.

Current scope:

- Public GitHub repository README pages.
- Public GitHub `.md` and `.markdown` file pages.
- GitHub Issues, Pull Requests, Wikis, private repositories, and non-GitHub websites are not supported in this beta.

Privacy:

wowMD is local-first. The extension reading flow does not upload document content, sell data, track browsing history, read cookies, or access local files. When you open Better View, the extension may fetch the current public Markdown document from GitHub or raw.githubusercontent.com so it can render the reader in your browser.

The website includes an optional feedback form for messages, feature preferences, and an optional email address. That website feedback flow is separate from the extension's reading features.
